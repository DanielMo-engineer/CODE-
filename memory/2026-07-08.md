# 2026-07-08 — v2.0 大赛功能改造

## 新增功能
1. **系统仪表盘** (`/dashboard`)
   - CPU/内存/磁盘/温度实时指标卡片（毛玻璃暗色）
   - BPU 信息、检测帧率、人数、运行时长
   - CPU+内存+温度 实时折线图（Chart.js，每秒刷新，60s 窗口）
   - 告警计数、进程数统计
   - 网络信息面板

2. **告警记录** (`/alerts`)
   - 摔倒事件持久化（alerts.json）
   - 待确认/已确认状态管理
   - 批量全部已读
   - 统计（全部/未处理/已确认）
   - 时间、人数、FPS 详细信息
   - 分页加载

3. **系统架构页** (`/about`)
   - 项目简介、软件栈分层架构图
   - RDK X5 硬件规格 + 实时性能指标
   - 关键技术决策说明（单模型姿势+摔倒、三级过滤、无代理直连、边缘+云端混合、PWA+mDNS）
   - 项目文件结构
   - 未来 Roadmap
