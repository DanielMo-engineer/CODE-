# RDK X5 人体摔倒检测系统 — 带姿态骨骼点版

基于 GS130W 双目摄像头 + **YOLO11n-pose** 的实时人体摔倒检测系统。

**区别于纯检测版**：单模型同时输出人体框 + 17个COCO关键点，在画面上绘制彩色骨架。

## 系统架构

```
GS130W Camera (1088x1280 @ 30fps)
        │
        ▼ NV12 → BGR
┌──────────────────────────┐
│  YOLO11n-pose (640x640)   │  ← 单模型：检测 + 姿态
│  - 人体检测框              │     17个关键点：
│  - 17个COCO关键点          │     头/肩/肘/腕/髋/膝/踝
│  - BPU 双核加速            │
└────────┬─────────────────┘
         │ boxes + keypoints
         ▼
┌──────────────────┐
│  FallDetector     │  ← 三级过滤（与原版一致）
│  (3-tier logic)   │     → ALARM / NORMAL
└──────────────────┘
```

## 关键点（COCO 17点）

```
 0: 鼻子      1: 左眼      2: 右眼
 3: 左耳      4: 右耳      5: 左肩
 6: 右肩      7: 左肘      8: 右肘
 9: 左手腕   10: 右手腕   11: 左髋
12: 右髋     13: 左膝     14: 右膝
15: 左踝     16: 右踝
```

**颜色编码**：
| 部位 | 颜色 | 关键点 |
|------|------|--------|
| 头部 | 青色 | 0-4 (鼻/眼/耳) |
| 肩部 | 橙色 | 5-6 (肩) |
| 手臂 | 亮绿 | 7-10 (肘/腕) |
| 躯干 | 深蓝 | 11-12 (髋) |
| 腿部 | 品红 | 13-16 (膝/踝) |

**骨架连线**：10条连线覆盖全身（手臂×2、躯干×2、腿部×2、肩部、髋部、头部）

## 使用方法

### 基本运行

```bash
cd /root/.openclaw/workspace/fall_detection
sudo bash run_fall_detection.sh
```

### 查看实时画面

打开浏览器访问：`http://设备IP:8080`

画面特点：
- 绿色框 = 正常 / 红色框 = 摔倒报警
- 白色角标装饰增强视觉效果
- 彩色骨架跟随人体动作
- 顶部半透明状态栏

### 常用参数调整

```bash
# 调整关键点显示数量
sudo bash run_fall_detection.sh --kpt-conf 0.5   # 只显示高置信度点（更干净）
sudo bash run_fall_detection.sh --kpt-conf 0.1   # 显示所有可能点（更灵敏）

# 显示关键点编号（调试用）
sudo bash run_fall_detection.sh --show-keypoint-labels

# 提高 web 清晰度
sudo bash run_fall_detection.sh --web-disp-w 800

# 修改检测灵敏度
sudo bash run_fall_detection.sh --score 0.3      # 降低灵敏度（减少误报）
sudo bash run_fall_detection.sh --score 0.2      # 提高灵敏度（检测更小的人）
```

### BPU 核心分配

默认使用双核 `--bpu-cores 0 1` 加速，如果和其他 BPU 应用冲突，可改为单核：

```bash
sudo bash run_fall_detection.sh --bpu-cores 0
```

## 与原版的区别

| 特性 | 纯检测版 | 姿态骨骼版（当前） |
|------|---------|-------------------|
| 检测模型 | YOLOv5s | YOLO11n-pose |
| 关键点 | ❌ 无 | ✅ 17个COCO关键点 |
| 骨架绘制 | ❌ 无 | ✅ 彩色连线 + 圆点 |
| 边框样式 | 普通矩形 | 双层框 + 角标装饰 |
| 状态栏 | 简单文字 | 半透明+彩色状态栏 |
| 指标面板 | 仅有文本 | 半透明+多行面板 |
| Web推流 | 480p | 640p（更高清） |
| BPU核心 | 单核 | 双核 |
